# 從麥塊看人類食物獲取進程

這是一個可用 GitHub Pages 架設的靜態網站。

網站入口：

- `index.html`

主要素材資料夾：

- `圖片/`

GitHub Pages 設定建議：

1. 在 GitHub 建立一個新的 repository。
2. 上傳 `index.html`、`styles.css`、`.nojekyll`、`README.md` 和整個 `圖片/` 資料夾。
3. 到 repository 的 `Settings` → `Pages`。
4. Source 選 `Deploy from a branch`。
5. Branch 選 `main`，資料夾選 `/root`。
6. 儲存後等待 GitHub 產生網址。
